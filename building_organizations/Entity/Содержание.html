Система управления строительством

Справочники

Город

Хранит информацию о городах.

  • ID - идентификатор города, основной ключ.
  • city_name - название города.

Улица

Хранит информацию об улицах.

  • ID - идентификатор улицы, основной ключ.
  • street_name - название улицы.

Банк

Хранит информацию о банках.

  • ID - идентификатор банка, основной ключ.
  • name - название банка.

Поставка

Хранит информацию о поставках.

  • ID - идентификатор поставки, основной ключ.
  • supplier_id - внешний ключ, ссылается на таблицу поставщиков.
  • delivery_date - дата поставки.

Вид работы

Хранит информацию о видах работ.

  • ID - идентификатор типа работы, основной ключ.
  • name - название типа работы.

Измерение стройматериалов

Хранит информацию о единицах измерения строительных материалов.

  • ID - идентификатор единицы измерения, основной ключ.
  • name - название единицы измерения.

Специализация

Хранит информацию о специализациях рабочих.

  • ID - идентификатор специализации, основной ключ.
  • name - название специализации.

Материалы и сотрудники

Стройматериалы

Хранит информацию о строительных материалах.

  • ID - идентификатор строительного материала, основной ключ.
  • name - название материала.
  • unit_id - внешний ключ, ссылается на таблицу единиц измерения.

Бригада

Хранит информацию о бригадах.

  • ID - идентификатор бригады, основной ключ.
  • name - название бригады.

Рабочие

Хранит информацию о рабочих.

  • ID - идентификатор рабочего, основной ключ.
  • last_name - фамилия.
  • first_name - имя.
  • middle_name - отчество.

Строительство

Объект

Хранит информацию о строительных объектах.

  • ID - идентификатор объекта, основной ключ.
  • object_name - название объекта.
  • house_number - номер дома.
  • city_id - внешний ключ, ссылается на таблицу города.
  • street_id - внешний ключ, ссылается на таблицу улицы.

Работа на объекте

Хранит информацию о работах на объекте.

  • ID - идентификатор работы, основной ключ.
  • date_ - дата отчета.
  • value_work - количество выполненных работ.
  • work_type_id - внешний ключ, ссылается на таблицу типов работ.
  • object_id - внешний ключ, ссылается на таблицу объектов.

Поставщик

Хранит информацию о поставщиках.

  • ID - идентификатор поставщика, основной ключ.
  • last_name - фамилия.
  • first_name - имя.
  • middle_name - отчество.
  • account_number - номер аккаунта.
  • inn - ИНН.
  • bank_id - внешний ключ, ссылается на таблицу банков.

Заявка

Хранит информацию о заявках.

  • ID - идентификатор заявки, основной ключ.
  • date_ - дата заявки.
  • material_id - внешний ключ, ссылается на таблицу материалов.
  • quantity - количество материала.
  • supplier_id - внешний ключ, ссылается на таблицу поставщиков.
